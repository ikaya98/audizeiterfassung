Google Sheets Verbindung
Schritt-für-Schritt Anleitung
1. Gehe zu sheets.google.com → neue leere Tabelle erstellen
2. Menü: Erweiterungen → Apps Script
3. Alles löschen, dann den google_apps_script.js Code einfügen
4. Auf Speichern klicken (Disketten-Symbol)
5. Auf Bereitstellen → Neue Bereitstellung klicken
6. Typ: Web-App wählen
7. Ausführen als: Ich — Zugriff: Jeder
8. Bereitstellen klicken → URL kopieren
9. URL unten einfügen und verbinden ✓
